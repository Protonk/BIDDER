# interlock/

Experimental wing for the interlocking-defects programme described in
`interlock/INTERLOCKING-DEFECTS.md`. Each subdirectory is a self-
contained run of one of the questions listed under §"What's open" in
that anchor doc.

## Subdirectories

- `game-start/` — first-digit fingerprint of `A_⟨3,5⟩` against primes
  and the `M_n` baselines (`INTERLOCKING-DEFECTS.md` §"What's open"
  item 1).
